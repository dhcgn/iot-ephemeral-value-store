Welcome to the IoT Ephemeral Value Store Server

This service allows you to temporarily store and retrieve IoT data via HTTP. Below are some examples of how you can interact with the server:

Getting Started

Just use this url generated just for you: /{{.UploadKey}}/?name=value and access it with /{{.DownloadKey}}/plain/name or in json with /{{.DownloadKey}}/json

Server Settings

  • Data rentention: {{.DataRentention}}

API Usage

  • Create Key Pair: GET /kp
  • Upload Data: GET /{upload-key}/?temp=23&hum=43
  • Download Data as JSON: GET /{download-key}/json
  • Download Specific Data Field: GET /{download-key}/plain/{param}

API Example, with static keys!

  • Upload Data: /5347d61e218f9a7988292c82ddfe347a276fcb82dd2df8aaad733c8970379f41/?temp=23&hum=43
  • Download Data as JSON: /39077ff759b3bbd0592728cbd16aab9e061328bbbffb740923337b23fdd0ad39/json
  • Download Specific Data Field:/39077ff759b3bbd0592728cbd16aab9e061328bbbffb740923337b23fdd0ad39/plain/hum

The download key can be derived from the upload key. echo -n "5347d61e218f9a7988292c82ddfe347a276fcb82dd2df8aaad733c8970379f41" | sha256sum
